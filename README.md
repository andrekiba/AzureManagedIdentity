# Azure Managed Identity

Chiavi e segreti sono una costante quando si lavora con le risorse in Azure, e la loro gestione può creare problemi sotto diversi punti di vista.
Vediamo come semplificarci la vita utilizzando Azure Managed Identity.
Attraverso diversi esempi pratici, esploreremo come usare vari servizi di Azure senza dover salvare informazioni sensibili da nessuna parte.
